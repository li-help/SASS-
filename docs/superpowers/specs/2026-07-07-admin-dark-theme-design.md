# Admin 暗色奢华主题美化 设计文档

## 概述

将 sass-kb-admin 管理后台从默认 Ant Design 浅色模板升级为暗色奢华风格，以深墨蓝为底、古铜金为强调色，打造高端企业级视觉体验。

## 色调体系

| 层级 | 颜色 | 用途 |
|------|------|------|
| 侧边栏 | `#080F1A` 深墨蓝 | 最深，视觉锚点 |
| 内容区背景 | `#0F1923` 暗蓝灰 | 主画布 |
| Header / 面包屑条 / 标签栏底 | `#162231` 中暗蓝 | 浮起一层 |
| 卡片 / 模态框 | `#1C2B3F` 浅暗蓝 | 浮起两层 |
| 强调色 | `#C8963E` 古铜金 | 选中态、图标、主按钮 |
| 辅助金 | `#E8C97A` 淡金 | hover、高亮边框 |
| 文字主 | `#E8ECF1` 暖白 | 正文 |
| 文字辅 | `#8FA0B8` 蓝灰 | 描述、提示 |
| 成功 | `#5B9A4B` | 深绿适配 |
| 警告 | `#D4A843` | 深金适配 |
| 错误 | `#D1493F` | 深红适配 |

## 各区域设计

### 侧边栏
- 背景 `#080F1A`，Logo 区金色图标
- 菜单项选中：左侧 3px 金色竖线 + 金色文字 + 微金背景 (`rgba(200,150,62,0.12)`)
- 菜单项 hover：微光从左扫入
- 折叠按钮金色

### Header
- 暗色半透明底 + 毛玻璃 (`backdrop-filter: blur(12px)`)
- 金色通知 badge
- 用户头像金色细边框

### 面包屑
- 小字号、蓝灰色，hover 变金色
- 分隔符金色

### 标签栏
- 深色背景，激活标签底部 2px 金色指示条
- 关闭 × hover 变金色
- 未激活标签文字灰色

### 内容卡片
- 暗色底 + `1px solid rgba(200,150,62,0.10)` 微金边框
- hover：边框金色增强 + 微上浮 (`translateY(-2px)`) + 阴影加深

### 统计卡片（Dashboard）
- 图标背景降饱和度适配暗色
- 数值金色或暖白
- hover 同上

### 表格
- 深色表头 `#162231`
- 行 hover 微金背景 `rgba(200,150,62,0.06)`
- 分页器暗色 + 金色激活页码

### 按钮
- 主按钮：金色实底 + 白色文字
- 默认按钮：暗色边框 + 蓝灰文字，hover 金色边框
- 危险按钮：红色降饱和

### 表单
- Input/Select：暗色底 + 微金边框
- focus 态：金色边框 + 微金外发光

## 实现方式

- Ant Design 6 `ConfigProvider theme` 的 `token` + `components` + `algorithm: theme.darkAlgorithm`
- 在 `darkAlgorithm` 基础上通过 token 覆盖注入金色强调色
- 少量自定义 CSS（侧边栏金线、毛玻璃、hover 浮起效果）通过 `index.css` 或组件级样式

## 涉及文件

| 文件 | 操作 |
|------|------|
| `App.tsx` | 主题配置重写，启用 darkAlgorithm + 金色 token |
| `index.css` | 全局暗色重置 + 自定义滚动条 + 毛玻璃等工具类 |
| `layouts/MainLayout.tsx` | 侧边栏金色线、Header 毛玻璃、面包屑/标签栏配色 |
| `pages/dashboard/index.tsx` | 统计卡片图标背景降饱和 |
